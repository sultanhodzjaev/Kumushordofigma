# KUMUSH ORDO - Silver Jewelry Store

## Overview
A React-based e-commerce frontend for handmade silver jewelry. Built with Vite, React 18, Tailwind CSS v4, and various UI component libraries.

## Tech Stack
- **Frontend Framework**: React 18.3.1
- **Build Tool**: Vite 6.3.5
- **Styling**: Tailwind CSS 4.1.12, Emotion, MUI
- **UI Components**: Radix UI, shadcn/ui patterns
- **Routing**: React Router v7
- **State/Forms**: React Hook Form
- **Charts**: Recharts

## Project Structure
```
/
├── index.html          # Entry HTML
├── src/
│   ├── main.tsx       # React entry point
│   ├── app/           # Application components
│   └── styles/        # CSS styles
├── vite.config.ts     # Vite configuration
├── postcss.config.mjs # PostCSS config
└── package.json       # Dependencies
```

## Development
- **Port**: 5000 (frontend only)
- **Command**: `npm run dev`
- **Build**: `npm run build`

## Deployment
- Static deployment via Vite build
- Output directory: `dist`
